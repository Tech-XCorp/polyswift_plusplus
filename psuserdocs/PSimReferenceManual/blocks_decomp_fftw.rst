Slab (FFTW)
------------------------------

:command:`fftw`:
    kind of :command:`Decomp` defines a decompostion for the FFTW slab configuration
    
fftw Parameters
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

:option:`transposeFlag` (string , default = 'off'):
    string flag for selecting the NORMAL or TRANSPOSE layout for the decomp
